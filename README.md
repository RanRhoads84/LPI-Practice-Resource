# LPI Linux Essentials Practice

An interactive, browser-based practice quiz for the **LPI Linux Essentials (010-160) v1.6** certification exam. No build step, no dependencies — open the HTML file and go.

## Features

- **Instant feedback** — each answer is marked correct/incorrect immediately, with the correct answer revealed
- **All five exam topics** — filter by any combination of topics or practice them all at once
- **Flexible session length** — 10, 20, 40 questions, or the full question bank
- **Exam-weighted mode** — selecting all topics with 40 questions mirrors the real exam's 7/9/9/8/7 topic distribution
- **Results summary** — per-topic breakdown and pass/fail indicator at the 66 % threshold
- **Keyboard navigation** — answer with `1`–`4`, advance with `Space`/`Enter`
- **Zero setup** — single self-contained HTML file, works offline

## Usage

```
# Clone the repo
git clone https://github.com/<your-username>/LPI-Practice-Resource.git

# Open in any modern browser — no server required
open lpi-essentials-practice.html   # macOS
start lpi-essentials-practice.html  # Windows
xdg-open lpi-essentials-practice.html  # Linux
```

Or just download [`lpi-essentials-practice.html`](lpi-essentials-practice.html) directly and open it.
